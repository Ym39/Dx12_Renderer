　ままま式改変あぴミク SeasonII シリーズをダウンロードいただき、ありがとうございます。

【Season-II の特徴】
１　あぴミクが即売会で手作り感あふれるコスプレ売り子をしてくれているようなシチュエーションをイメージしています。
２　素体を一新しました。概要は当分の間、https://yayoisakura.github.io/mmd/ で公開しています。（怪しい英語だけど）。
３　MMD用に残しておいたwindowsマシンが寿命になってしまったので、以下の点を断念して公開に踏み切っています。
・ボディのメッシュのつなぎ目の補正（通常の使用では問題ありませんが、シェーダーの設定によっては腕やわき腹で色の断層が出来てしまうかも）
・一部のモデルで貫通の修正。（モーションによっては脇の下（天津風）、背中（ハロウィンリベッチオ）が怪しくなります）


ご利用なさる前に、ご面倒とは存じますが利用条件をお読みいただきますよう、お願い申し上げます。
最後になりますが、救世主とも言うべき各種お役立ちツールの作者様、極北P様、そぼろP様、(´・ω・｀)うｐ様には、この場を借りて深くお礼申し上げます。
